HTML简介

什么是HTML？

HTML是用来描述网页的一种语言。
1.超文本标记语言—–Hyper Text Markup Language
2.它不是编程语言，是一种标记语言（markup language）
3.标记语言是一套标记标签（markup tag）
4.HTML使用标记标签来描述网页
